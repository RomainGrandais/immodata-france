ImmoData France API

Données immobilières françaises agrégées en temps réel pour les agents IA.

BAN · DVF · DPE · INSEE Claude Haiku MCP compatible OpenAPI 3.1 Node 18+ · Vercel

Authentification

Toutes les requêtes (sauf /health) nécessitent une clé API. Passez-la dans le header x-api-key.

curl https://immodata-france.vercel.app/v1/bien \
  -H "x-api-key: votre_clé" \
  --get --data-urlencode "adresse=12 rue de la Paix Lyon"

Vous pouvez aussi utiliser Authorization: Bearer votre_clé ou le query param ?apikey= (déconseillé — apparaît dans les logs).

Analyse immobilière

GET/v1/bien

Géocode l'adresse via la BAN puis agrège DVF, DPE ADEME et INSEE en un seul JSON. Génère un résumé Claude Haiku si format=ai.

Paramètres

ParamètreTypeRequisDescription
adressestringOuiAdresse textuelle française (5–200 caractères)
formatstringNonai (défaut) ou raw (sans résumé, zéro coût LLM)

Essayez

Exemple cURL

curl "https://immodata-france.vercel.app/v1/bien?format=raw" \
  -H "x-api-key: votre_clé" \
  --get --data-urlencode "adresse=12 rue de la Paix Lyon"

Configuration MCP pour Claude

Ajoutez ce bloc dans votre configuration Claude Code (~/.claude/claude.json ou paramètres MCP) :

{
  "mcpServers": {
    "immodata": {
      "type": "http",
      "url": "https://immodata-france.vercel.app/mcp",
      "headers": {
        "x-api-key": "votre_clé"
      }
    }
  }
}

L'outil rechercher_bien sera alors disponible dans toutes vos conversations Claude.
